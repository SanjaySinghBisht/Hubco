All About IEC Code Registration in India

What is a Import Export Code (IEC)?

Import Export Code (IEC) is the 10 diigt code issed by DGFT office in India. IEC code registration is mandatory for every exporter importer to do the business of import export legitimately.

How to register Import Export Code (IEC)?

Import Export Code (IEC) is mandatory for all exporters and importer in India as per the Foreign Trade Policy. All the registration steps has been broadlu categorised under three sub heads:

#Apply for DSC: The IEC registration process is completly online and hence needs to be signed by a digital Signature (DSC). DSC is a tool which is used to sign on computers.

#Fill online Application & Upload Doc: After applying for DSC, one must fill the application and upload the documents properly to avoid any mistake. A small mistake will also leads to rejection of application.

#Upload and Follow up: Once application is filed properly and submitted then you must follow up in case IEC is not issued within time limit.

Why to register a Import Export Code (IEC)?

If you want to start a import export business, then you must have a IEC code otherwise you cannot import or export on your own. Further if you want to sell on Amazon.com, then also you should apply for IEC code.

Requirements for IEC Code Registration

There is only one requirement for IEC code in India i.e. your business name should be printed on the copy of cancelled cheque. If it is not printed then your application will be liable to be rejected.Here are some other few important points:

  • - IEC code can be applied on savings bank account, however it is recommended to have current account.
